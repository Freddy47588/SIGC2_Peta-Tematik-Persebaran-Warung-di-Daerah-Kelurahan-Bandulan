Abstrak

Ringkasan penelitian/pemetaan

Pemetaan persebaran warung di Kelurahan Bandulan bertujuan untuk memvisualisasikan lokasi warung dalam konteks jaringan jalan dan penggunaan lahan. Hasil pemetaan disajikan dalam bentuk layout peta serta web peta interaktif, sehingga pengguna dapat mengeksplorasi sebaran warung, memahami pola kedekatan dengan ruas jalan utama maupun gang, dan mengidentifikasi area permukiman yang padat layanan. Informasi ini dapat dimanfaatkan untuk dukungan perencanaan wilayah, aksesibilitas layanan, serta kebutuhan kajian akademik berbasis GIS.

Tujuan

  1. Memetakan dan menampilkan titik persebaran warung di Kelurahan Bandulan.
  2. Menunjukkan hubungan lokasi warung dengan jaringan jalan (desa/perumahan/gang).
  3. Menyediakan media eksplorasi berbasis web untuk akses informasi spasial secara cepat.
  4. Mendukung analisis sederhana terkait layanan ekonomi mikro di wilayah permukiman.
